# TP Final React — [Tu Nombre]

App hecha con Vite + React que usa React Router y PokeAPI.
Muestra un listado de pokemones (nombre + imagen) y un detalle por id o nombre.

# Tecnologías

- Vite + React
- React Router DOM
- PokeAPI (https://pokeapi.co)

# Scripts

```bash
npm install
npm run dev
# opcional:
npm run build
npm run preview
```
